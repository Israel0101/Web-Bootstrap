Counter Strike 1.6

Este site é uma simples homenagem ao jogo que marcou não só a minha infância, mas também, a de milhares de gamers.

Sobre o jogo:

Counter-Strike (também abreviado por CS) é um popular jogo eletrônico de tiro em primeira pessoa. O jogo é baseado em rodadas nas quais equipes de contraterroristas e terroristas combatem-se até a eliminação completa de um dos times, e tem como objetivo principal plantar e desarmar bombas, ou sequestrar e salvar reféns.

DICA antes de jogar...

Antes de iniciar o game, dê uma configurada em sua mira, isso ajudará muito no desempenho. Instruções: Para fazer com que a mira fique pequena no jogo cs, primeiro abra o Counter Strike 1.6 e digite ASPAS, para abrir o console do jogo. Digite os comandos ao lado.

Atualizados 3 minutos atrás

Lembre-se de que você deve digitar um comando de cada vez no console do jogo

cl_lw 1 / cl_lc 1 / sv_aim 0 / cl_crosshair_size pequeno / cl_crosshairscale 2500 / cl_dynamiccrosshair 0 / dynamic_crosshair 0 / cl_dynamic_crosshair 0 / cl_obervecrosshair 0 /
